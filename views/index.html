Welcome To My Demo Server

API Endpoint: https://expressjs-rest-api-without-database.onrender.com/users

  • GET: To get all the users.
  • GET a User: To get a single user. Example: https://expressjs-rest-api-without-database.onrender.com/users/{id}
  • POST: To create a new user. Send the data in the body using the application/x-www-form-urlencoded format.
  • PUT: To Update a user. Example: https://expressjs-rest-api-without-database.onrender.com/users/{id}
  • Delete: Delete a User. Example: https://expressjs-rest-api-without-database.onrender.com/users/{id}
